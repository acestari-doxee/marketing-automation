## 🚨 Common issues

**The `.command` file won't open / "developer cannot be verified"**  
→ Right-click the file → click **Open** → click **Open** again in the dialog.

**"Python not found" or "command not found"**  
→ Python is not installed, or was not added to PATH during installation. Reinstall Python and make sure to check **"Add Python to PATH"**.

**"Authentication error" / "Invalid API key"**  
→ Check that `config.json` exists (not `config.example.json`) and that the keys are entered correctly, with no extra spaces.

**The black window opens and closes immediately**  
→ There's an error. Open **Terminal** (search "Terminal" on Mac), drag the `Start.command` file into the window, press Enter. You'll see the error message.
